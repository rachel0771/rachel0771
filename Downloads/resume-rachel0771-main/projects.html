My Projects

Pet Info

Pet Info is a personal project that I developed using the Pug templating engine. It serves as a comprehensive pet information registration and query system.

The application allows users to register details like their pets' names, select breeds, and record their pets' preferences. This functionality is particularly useful for pet owners who need to keep track of various aspects of their pet care.

The app connects to MongoDB, ensuring efficient data management and storage. Its intuitive interface design makes it a convenient and efficient platform for managing pet information.

Bubble Tea Review

Bubble Tea Review is a group project where we focused on creating a platform for reviewing Happy Lemon bubble tea beverages. The project was developed to emphasize user interaction and feedback.

It features a user-friendly interface where customers can log in via IAP to rate and review teas. Users can also view comments based on specific tea categories, making it easy to find reviews of particular interest.

Additionally, the project utilizes advanced data analysis techniques. This provides users with personalized recommendations, enhancing the user experience and fostering a more engaged community.
